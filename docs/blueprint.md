# **App Name**: AI BMI Analyzer

## Core Features:

- BMI Calculation: Calculate BMI based on user-provided height and weight.
- AI-Powered BMI Analysis: Use a generative AI tool to analyze the calculated BMI and provide personalized health recommendations and lifestyle tips.
- User Input Forms: Collect height and weight using input fields.
- Result Display: Show BMI calculation and AI analysis on the results page.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to create a calm and trustworthy feel.
- Background color: Very light blue (#F0F8FF), almost white, to ensure readability and a clean aesthetic.
- Accent color: Soft green (#98FB98) to represent health and well-being.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern yet warm feel.
- Use simple, line-based icons to represent different health metrics.
- Clean and intuitive layout, focusing on ease of use and readability.
- Subtle animations to enhance user interaction and provide visual feedback.
